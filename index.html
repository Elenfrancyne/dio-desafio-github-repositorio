Diferença entre Git e Git Hub

A principal diferença é que o Git se trata de um repositório local, ou seja, o software de versionamento instalado em seu computador, notebook, tablet, etc e o GitHub trata-se de um repositório remoto que está na rede, como se o Git Hub fosse uma nuvem que versiona seu projeto.

Como Criar um repositório no GitHub?

Abaixo seguem os tópicos do passo a passo para realizar a Criação de seu Repositório:

  • Abra o seu GitHub;
  • Acesse a guia Repositório/Repositories;
  • Selecione o botão Novo/New;
  • Insira o nome do seu Repositório
  • Descreva o seu projeto;
  • Escolha entre Público/Public e Privado/PrivatePúblico/Public e Privado/Private;
  • Marque as opções necessárias para criar o seu projeto de acordo com a sua necessidade "Adicionar um arquivo README/ Add a README file", "Adicionar .gitignore/Add .gitignore" e "Escolha uma licença/Choose a license", aconselho a adicionar no minimo um arquivo README
  • Clique em Criar Repositório/ Create Repository;

Como Clonar um repositório no GitHub( remoto) para um repositório local?

Abaixo seguem os tópicos do passo a passo para Clonar o seu Repositório:

  • Após a Criação do seu repositório, volte a tela inicial do seu GitHub, acesse novamente a aba Repositório/ Repositories e clique no repositório Criado
  • Você verá um botão verde escrito Código/ Code clique neste botão;
  • Escolha a sua guia preferida de acordo com sua necessidade "HTTPS", "SSH", "GitHub CLI", eu uso o HTTPS;
  • Abaixo da guia aparecera o link referente ao seu projeto e ao lado desse link tera a opção de copiar, selecione a opção de copiar( observação é o desenho de dois quadrados, como uma folha sobre a outra igual do Word);
  • Crie uma pasta em seu computador para receber esse repositório;
  • Após a criação da pasta clique sobre ela com o botão contrário do usual, e selecione i>"Git Bash Here" ;
  • Neste momento irá abrir o terminal do Git;
  • Digite o comando "git clone" e cola o endereço do seu repositório que foi copiado la no GitHub depois da enter, seu repositorio sera clonado;
  • Digite o comando "cd + nome da pasta que foi clonada" neste momento você poderá trabalhar dentro do repositório clonado, fique atento para escrever o nome correto da pasta clonada;
  • Confira se deu tudo certo através do comando "git status" e o enter;
  • Neste momento voce podera acessar as pastas no seu Pc, e criar os arquivos normalmente, conforme sua necessidade;
  • Quando vc acessar o Git bash, digite novamenteo comando "git status" e verifique as adições realizadas.

Como commitar e dar push em um repositório do Git(local) para o GitHub( remoto) ?

Abaixo seguem os tópicos do passo a passo para Commitar o seu Repositório:

  • Após digitar o comando "git status" e verioficar que há adiçoes para serem realizadas, vocÊ irá digitar o comando "git add." ou "git add -a" e dá um enter, assim você irá adicionar as pastas e arquivos criados;
  • Digite novamente o comando "git status" , para verificar se tudo está certo;
  • Digite o comando "git commit -mt" e adicione uma mensagem explicando a comitação a mensagem deve ser escrita entre aspas"" e após o comando digitado, após finalizar a mensagem aperte o botão enter;
  • Digite novamente o comando "git status" , para verificar se tudo está certo;
  • Agora digite o comando "git push origin main" assim poderá finalizar o processo de commit e push e podera ver o seu projeto tanto no repositório remoto, como tambem no repositório local, com as mesmas atualizações
